Project 2

Title : Automated plant watering system using Arduino uno

Duration: 1 week

Project Description :

• Developed and implemented an automatic plant watering system using Arduino Uno microcontroller.

• Designed to provide precise and customizable irrigation to potted plants,
the system incorporates moisture sensors to detect soil moisture levels and activates a water pump accordingly.

• Utilized Arduino programming language to create algorithms for sensor readings and pump control,
ensuring efficient water usage and plant health.

Key Contributions:

• Designed system architecture and selected components based on project requirements.

• Programmed Arduino Uno microcontroller to interface with moisture sensors and control water pump operation.

• Conducted rigorous testing and calibration to optimize system performance and reliability.

• Implemented user-friendly interface for adjusting watering schedules and sensor thresholds.

• Documented project specifications, codebase, and assembly instructions for future reference and replication.

Impact:

• Provided an innovative solution for automated plant care, catering to individuals with busy schedules or limited access to water.

• Demonstrated proficiency in hardware prototyping, sensor integration, and microcontroller programming.

• Enhanced understanding of electronics, automation, and software development through hands-on project experience.

• Contributed to environmental sustainability by promoting efficient water usage and supporting plant growth and well-being.
